Description of the Attack

  • Code only present in a range of versions of deployed software
    • Malware transplanted into Orion build process
    • Code eventually found on a crashed server with a build VM
  • Attacker did recon prior to attack
    • Probed vulnerabilities
    • Removed all traces

Stealthy code

  • Small Footprint
  • Obfuscated by de-compiling and re compiling
  • Made to look like build-type source code
  • Features to look for Malware monitoring/antivirus
  • Patient to evade logging
